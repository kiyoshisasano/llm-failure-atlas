You are a diagnostic signal extractor for LLM agent failure analysis.

Given a conversation log between a user and an AI agent, determine whether each of the following signals is present. Answer ONLY with a JSON object containing boolean values.

Signals to evaluate:

1. ambiguity_without_clarification
   TRUE if: The user's request was ambiguous AND the agent did NOT ask for clarification before proceeding.
   FALSE if: The request was clear, OR the agent asked a clarifying question.

2. assumption_persistence_after_correction
   TRUE if: The user corrected the agent, but the agent continued with its original (incorrect) assumption.
   FALSE if: No correction occurred, OR the agent adjusted after correction.

3. no_hypothesis_branching
   TRUE if: The agent committed to a single interpretation without considering alternatives.
   FALSE if: The agent considered multiple possibilities.

---

Conversation log:

{conversation_log}

---

User query: {query}
Agent response: {response}
User feedback (if any): {feedback}

---

Respond ONLY with this JSON format, no other text:

{
  "ambiguity_without_clarification": true or false,
  "assumption_persistence_after_correction": true or false,
  "no_hypothesis_branching": true or false
}
